# Aidat Borcu Ödeme Sistemi

### Projenin Amacı
İnternet üzerinden kişilerin apartmana dair olan tüm ödemelerin yapılabilmesini sağlar.
Apartmana ait tüm bilgi ve duyuruların sistemde var olmasınına olanak sağlar.
İletişimin kolay bir şekilde yürütülmesi sağlanır.

### **Projede Kullanılan Teknolojiler**

- HTML, CSS
- PHP
- MySQL
